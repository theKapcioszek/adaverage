# adaverage  

Small program for showing arithmetic average, dominant and median of given array of numbers  

## Quick Start:  

```console
$ gnatmake ./src/adaverage.adb
$ ./adaverage <array of numbers>
```  
Example Usage:  

```console
$ ./adaverage 1 2 2 2 5 5 6
```  

Expected Output:  

```console

Average:
      3.286
Dominant:
      2.000
Median:
      2.000

```
